Rake tool

  • Rake is a ruby automation tool
  • By default Rake search current folder for a file called "Rakefile"

create rake file

 put ruby code into a file called Rakefile

rake file example

desc "rake testing" # help description
task :say_rake do # taks name
  puts "do rake task" # task code here
end

show rake all task description

$ rake -T

call rake task

$ rake [ task name ]

call multi rake task

desc "rake testing"
task :multi => [:task_1, :task_2] do
  puts "going to do multi"
end
