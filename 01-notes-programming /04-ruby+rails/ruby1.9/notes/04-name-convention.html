Name convention

name
_name		 Local variables, method parameters, and method names should all start with a lowercase letter or with an underscore.
$debug Global variables are prefixed with a dollar sign $
@name Instance variables begin with an “at” sign @
@@total Class variables start with two “at” signs @@
fish_and_chips Multiword instance variables are written with underscores between the words
price= Method names may end with the characters ?, ! and =
String Class names, module names, and constants must start with an uppercase letter.
MyClass Multiword class names are written in MixedCase (with each word capitalized).
