Ruby Gem

  • RubyGems is a standardized packaging and installation framework for Ruby libraries and applications.
  • Developers bundle their applications and libraries into single files called gems. These files conform to a standardized format and typically are stored in repositories on the internet.
  • The RubyGems system provides a command-line tool, appropriately named gem, for manipulating these gem files. It also provides integration into Ruby so that your programs can access gems as libraries.
  • Prior to Ruby 1.9, it was your responsibility to install the RubyGems software on your computer. Now, however, Ruby comes with RubyGems baked right in.

Check current installed Gems

$ gem list

Install latest gem

$ gem install [ gem name ]
#If you’re running gem install on a Unix platform, you’ll need to prefix the command with sudo

Install specific gem by specific version

$ gem install builder version '< 2'

Load gem

$ require 'builder'

Load specific version of gem

gem 'builder', '< 1.0'
require 'builder'

Read gem help document

$ gem server # start server to show all gem help documents

$ Ctrl + c # stop gem server

Remove gem

$ gem uninstall [ gem name ]
