working with project

create project

 Select File>New Project
drop whole folder into text mate icon

In terminal, open folder as project
  1. mate . # a project of the current directory
  2. mate a.txt b.txt # a project of two files
  3. mate *.csv # a project of CSV files in the current directory

 

working with drawer

change drawer to another side
  1. Select View > Hide Project Drawer, if the drawer is currently visible.
  2. Move the project window close enough to the edge of the screen so the drawer doesn’t have room to open on the side you don’t want it on.
  3. Bring the drawer back with View > Show Project Drawer.

group files in drawer

You can also create groups for the files using the folder button at the bottom of the project drawer or by Apple -clicking several files and selecting Group Selected Files from the action menu at the bottom of the project drawer,

 

content editing

  

edit model

Freehanded Editing

  • Edit>Mode>Freehanded Editing
  • TextMate will allow you to place the caret freely with keyboard arrow movement or a mouse click. If you insert content, TextMate will add the needed whitespace to support the content at that location.

Overwrite Mode

  • Edit>Mode>Overwrite Mode
  • Instead of inserting text between characters as it usually does, TextMate will replace the characters above the caret as you type.

  • End of column alt + down

auto completion

Every chance you get, start typing the first few letters of something, and press "ESC". You can use this to finish off variables, methods, tags, or any other name already in your document. When there is more than one completion choice, you just keep pressing "ESC" to cycle through them.

Change letter case
  • Uppercase selection Ctrl+U
  • Lowercase selection Ctrl+Shift+U
  • Title case each word of the selection Ctrl+Alt+U
  • Reverse the case of the selection Ctrl+G
