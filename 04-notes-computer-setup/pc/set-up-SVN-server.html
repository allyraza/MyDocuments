Set up SVN on XP

01 install XP  
02 install SP2, SP3

WindowsXP-KB835935-SP2-ENU.exe

WindowsXP-KB936929-SP3-x86-ENU.exe
03 install SVN

using default setting

Setup-Subversion-1.5.6.msi
04 install Apache

using default setting

apache_2.2.14-win32-x86-openssl-0.9.8k.msi
05 create Repositories
  1. create folder on hard drive
    C:\Repositories\project1
  2. create repositories in command line
    svnadmin create C:\Repositories\project1
06 copy SVN models to Apache folder copy mod_authz_svn.so and mod_dav_svn.so from C:\Program Files\Subversion\bin to C:\Program Files\Apache Software foundation\Apache2.2\modules
07 update Apache config
  1. open C:\Program Files\Apache Software foundation\Apache2.2\conf\httpd.conf
  2. add
    LoadModule dav_module modules/mod_dav.so
    LoadModule dav_svn_module modules/mod_dav_svn.so
    LoadModule authz_svn_module modules/mod_authz_svn.so
  3. add to end
    Include c:/etc/subversion.conf
08 create a folder to save config file create a folder c:/etc
09 set user name and password

cd C:\Program Files\Apache Software Foundation\Apache2.2\bin
htpasswd -cm C:\etc\svn-auth-file harry
New password: *****
Re-type new password: *****
Adding password for user harry
10 set user w/r right

in file C:\etc\svn-acl

[project1:/]
harry = rw

 
11 set SVN config file

in file C:\etc\subversion.conf

<Location /project1>
DAV svn
SVNPath C:/Repositories/project1

AuthType Basic
AuthName "Subversion Project1 repository"
AuthUserFile c:/etc/svn-auth-file

Require valid-user

AuthzSVNAccessFile c:/etc/svn-acl
</Location>

  2. test localhost/project1
