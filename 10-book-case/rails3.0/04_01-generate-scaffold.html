Generate scaffold

Scaffold are major pieces of an application (based on MVC) generated by rails.

rails generate scaffold Post name:string title:string content:text #command to generate scaffold

class CreatePosts < ActiveRecord::Migration
  def change
    create_table :posts do |t|
      t.string :name
      t.string :title
      t.text :content
      t.timestamps
    end
  end
end

Model

app/models/post.rb

model, interactive with database

class Post < ActiveRecord::Base
  attr_accessible :content, :name, :title

    validates :name, :presence =>true #need name
    validates :title, :presence =>true, :length =>{ :minimum =>5 } #need title with min 5 letters
end

 

the validation code is added manually
